# User Management Application

This is a full-stack User Management application developed as part of a coding assignment.

The application allows:
- Creating a new user
- Viewing a list of existing users
- Importing users in bulk using a CSV file

The project is built with a clean separation of frontend and backend and can be run locally for demonstration.

---

## Tech Stack

- **Backend**: Spring Boot (Java 17), Spring Data JPA
- **Frontend**: React (Hooks)
- **Database**: PostgreSQL
- **Build Tools**: Maven, npm

---

## Features

- Create new user via UI
- View all users in a table
- Import users using CSV file
- REST APIs for user operations
- PostgreSQL database integration
- CORS enabled for frontend-backend communication
- Local environment demo ready

---

## Database Setup

1. Install PostgreSQL
2. Create database:
   CREATE DATABASE userdb;
